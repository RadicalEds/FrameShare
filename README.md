# FrameShare v0.1

Image Sharing Via Video Frames   
A small utility that packs images into a video file and vice versa, frame by frame   
Has an option to Obfuscate the Images by "Scrambling" their Pixels with a Numerical Seed   

## Requirements

- ffmpeg
- imagemagick
